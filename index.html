Bài tập JS buổi 1

Bài tập 1 : tính tiền lương nhân viên

INPUT :
  • Số ngày làm của nhân viên : n
  • Tiền lương mỗi ngày : 100.0000
PROCESS
  • Tạo biến x
  • Gán giá trị số ngày làm của nhân viên cho biến x
  • Tạo biến tiền lương y
  • Tính lương nhân viên theo công thức : y = x*100000
  • Xuât giá trị ra input
OUTPUT
  • Kết quả lương của nhân viên.

Bài tập 2 : tính giá trị trung bình của 5 số

INPUT :
  • Giá trị số thứ nhất
  • Giá trị số thứ hai
  • Giá trị số thứ ba
  • Giá trị số thứ tư
  • Giá trị số thứ năm
PROCESS
  • Tạo 5 biến : x1,x2,x3,x4,x5
  • Tạo biến giá trị trung bình : TBC
  • Gán giá trị của từng số người dùng nhập vào tương ứng cho 5 biến
  • Tính giá trị trung bình theo công thức : TBC = (x1+x2+x3+x4+x5) / 5
  • Xuât giá trị trung bình của 5 số ra input
OUTPUT
  • Giá trị trung bình của 5 số.

Bài tập 3 : Quy đổi tiền

INPUT :
  • Tổng tiền USD muốn quy đổi.
PROCESS
  • Tạo biến x
  • Gán giá trị tổng tiền quy đổi cho x
  • Tạo biến tổng tiền sau quy đổi y
  • Tính tiền quy đổi theo công thức : y = x*23500
  • Xuât giá trị của y ra input
OUTPUT
  • Tổng tiền sau khi quy đổi

Giá USD hiện tại : 23.500 VNĐ

Bài tập 4 : Tính chu vi & diện tích hình chữ nhật

INPUT :
  • Chiều dài hình chữ nhật
  • Chiều rộng hình chữ nhật
PROCESS
  • Tạo biến x,y và gán giá trị tương ứng là chiều dài và chiều rộng HCN
  • Tạo biến z1,z2 để lưu chu vi và diện tích HCN
  • Tính chu vi HCN : z1 = (x+y)*2
  • Tính diện tích HCN : z2 = x*y
  • Xuât giá trị z1,z2 ra input
OUTPUT
  • Chu vi và diện tích của HCN

Bài tập 5 : Tính tổng các chữ số của số 2 chữ số

INPUT :
  • Số có 2 chữ số
PROCESS
  • Tạo biến x và gán giá trị của số nhập vào cho x
  • Tạo biến y1,y2,z
  • Tách lấy số hàng chục của x theo công thức : y1 = x/10
  • Tách lấy số hàng đơn vị của x theo công thức : y2 = x%10
  • Tổng các chữ số của x theo công thức : z = y1+y2
  • Xuât giá trị của z ra input.
OUTPUT
  • Tổng các chữ số của số có 2 chữ số được nhập vào.

Bài tập JS buổi 2

Bài tập 1 : Sắp xếp thứ tự tăng dần của dãy số

INPUT :
  • Số nguyên thứ 1
  • Số nguyên thứ 2
  • Số nguyên thứ 3
PROCESS
  • Tạo biến x1,x2,x3
  • Gán giá trị của z,y,z tương ứng với số thứ 1, số thứ 2, số thứ 3
  • Tạo biến y1,y2,y3
  • Đi so sánh lần lượt x1, x2,x3 với nhau
  • Gán giá trị số lớn nhất cho y1
  • Gán giá trị số lớn thứ 2 cho y2
  • Gán giá trị số lớn thứ 3 cho y3
  • Xuât ra input theo thứ tự y3 y2 y1
OUTPUT
  • Xuất ra dãy số theo thứ tự từ bé đến lớn

Bài tập 2 : Đưa ra lời chào phù hợp với người sử dụng

INPUT :
  • Người sử dụng máy tính : bố, mẹ, anh trai, em gái.
PROCESS
  • Tạo biến x
  • Gán giá trị cho x tương ứng với người sử dụng.
  • So sánh giá trị của x với giá trị B,M,A,E
  • Đưa ra lời chào hỏi phù hợp với giá trị của x
  • Xuât lời chào hỏi ra input.
OUTPUT
  • Lời chào hỏi phù hợp với đối tượng người sử dụng

Bài tập 3 : Đếm số chẵn, số lẻ

INPUT :
  • Giá trị số thứ 1
  • Giá trị số thứ 2
  • Giá trị số thứ 3
PROCESS
  • Tạo biến x1,x2,x3 và gán giá trị tương ứng với giá trị số thứ 1, số thứ 2, số thứ 3.
  • Tạo biến y1,y2
  • Tạo biến z1,z2
  • Lần lượt lấy x1,x2,x3 chia cho 2
  • Nếu x1 hoặc x2 hoặc x3 chia hết cho 2 (x%2 === 0), z1 tăng thêm 1
  • Nếu x1 hoặc x2 hoặc x3 không chia hết cho 2 (x%2 !== 0), z2 tăng thêm 1
  • Xuât giá trị của z1,z2 ra input.
OUTPUT
  • Có bao nhiêu số chắn và số lẻ trong 3 số được nhập.

Bài tập 4 : Xác định tam giác

INPUT :
  • Giá trị cạnh thứ nhất
  • Giá trị cạnh thứ hai
  • Giá trị cạnh thứ ba
PROCESS
  • Tạo biến x1,x2,x3 và gán giá trị tương ứng với giá trị cạnh thứ 1, cạnh thứ 2, cạnh thứ 3.
  • Kiểm tra xem đây có phải là ba cạnh của tam giác: Tổng 2 cạnh luôn lớn hơn cạnh còn lại, hiệu 2 cạnh luôn nhỏ hơn cạnh còn lại
  • So sánh độ dài các cạnh
  • Có 1 cặp cạnh bằng nhau : tam giác cân
  • Độ dài 3 cạnh bằng nhau : tam giác đều
  • Độ dài 3 cạnh thỏa mãn định lý pytago : tam giác vuông
  • Xuất ra tính chất của tam giác
OUTPUT
  • Tính chất của tam giác : vuông, cân, đều.

Bài tập JS buổi 3

Bài tập 1 : Quản lý tuyển sinh

INPUT :
  • Giá trị điểm chuẩn
  • Giá trị điểm môn thi 1
  • Giá trị điểm môn thi 2
  • Giá trị điểm môn thi 3
  • Giá trị khu vực dự thi
  • Giá trị đối tượng dự thi
PROCESS
  • Tạo biến x1,x2,x3,x4,x5,x6
  • Gán giá trị điểm chuẩn cho x1
  • Gán giá trị điểm môn thi 1 cho x2
  • Gán giá trị điểm môn thi 2 cho x3
  • Gán giá trị điểm môn thi 3 cho x4
  • Gán giá trị khu vực dự thi cho x5
  • Gán giá trị đối tượng dự thi cho x6
  • Khai báo biến y,z
  • Giá trị y(điểm ưu tiên) = điểm của khu vực dự thi + điểm của đối tượng dự thi
  • Giá trị z(tổng điểm) = y + x2 + x3 + x4
  • So sánh giá trị của z với điểm chuẩn
  • Xuât ra kết quả tuyển sinh của thí sinh
OUTPUT
  • Kết quả tuyển sinh của thí sinh

Bài tập 2 : Tính tiền điện

INPUT :
  • Giá trị số điện sử dụng
PROCESS
  • Tạo biến x,y
  • Gán giá trị số điện sử dụng cho x1
  • So sánh x với bảng tính giá điện
  • x1 < 50 thì y = x1*500
  • x1 >= 50 && x1 < 100 thì y = 50*500 + (x1 - 50)*650
  • x1 >= 100 && x1 < 200 thì y = 50*500 + 50*650 + (x1 - 100)*850
  • x1 >= 200 && x1 < 350 thì y = 50*500 + 50*650 + 100*850 + (x1 - 200)*1100
  • x1 >= 350 thì y = 50*500 + 50*650 + 100*850 + 150*1100 + (x1 - 350)*1300
  • Xuất ra input giá trị tiền điện phải trả
OUTPUT
  • Tiền điện phải thanh toán

Bài tập 3 : Tính thuế thu nhập cá nhân

INPUT :
  • Giá trị tổng thu nhập hằng năm
  • Giá trị số người phụ thuộc
PROCESS
  • Tạo biến x,y,z
  • Gán giá trị tổng thu nhập hằng năm cho x
  • Gán giá trị số người phụ thuộc cho y
  • Tính số tiền chịu thuế(z) theo công thức ở đề bài
  • Tính thuế TNCN phải đóng
  • z < 60 thì TNCN = z*0.05
  • z > 60 && z <= 120 thì TNCN = z*0.1
  • z > 120 && z <= 210 thì TNCN = z*0.15
  • z > 210 && z <= 384 thì TNCN = z*0.2
  • z > 384 && z <= 624 thì TNCN = z*0.25
  • z > 624 && z <= 960 thì TNCN = z*0.3
  • z > 960 thì TNCN = z*0.35
  • Xuất ra input giá trị tiền thuế TNCN phải đóng
OUTPUT
  • Tiền thuế TNCN phải đóng

Bài tập 4 : Tính tiền cáp

INPUT :
  • Loai khach hang
  • Số kết nối (đối với khách hàng doanh nghiệp)
  • Số kênh cao cấp
PROCESS
  • Nêu khách hàng là cá nhân tính tiền cáp theo công thức
  • tienCap = 4.5 + 20.5 + kenhCaoCap*7.5
  • Nêu khách hàng là doanh nghiệp tính tiền cáp theo công thức
  • Nếu số kênh kết nối <= 10 : tính tiền cáp theo công thức
  • tienCap = 15 + 75*soKetNoi + 50*kenhCaoCap
  • Nếu số kênh kết nối > 10 : tính tiền cáp theo công thức
  • tienCap = 15 + 75*10 + 5*(soKetNoi - 10) + 50*kenhCaoCap
OUTPUT
  • Tiền cáp phải đóng của cá nhân hoặc của doanh nghiệp
